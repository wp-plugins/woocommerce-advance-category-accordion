== Installation ==

<ol style="margin: 0px 0px 22px 22px; padding: 0px; color: #444444; font-family: 'Open Sans', sans-serif; font-size: 13px; line-height: 22px;">
  <li style="margin: 0px; padding: 0px;">
    Upload the full plugin directory to your /wp-content/plugins/ directory or install the plugin through Plugins in the administration
  </li>
  <li style="margin: 0px; padding: 0px;">
    Activate the plugin through Plugins in the administration
  </li>
  <li style="margin: 0px; padding: 0px;">
    Have fun creating your first widget under the menu <strong><em>Appearance> </em>Widgets</strong> and drag the <strong>woo category accordion</strong> widget to your side bar.
  </li>
  <li style="margin: 0px; padding: 0px;">
    Optional:Set your option in the widget as your wish.
  </li>
  <li style="margin: 0px; padding: 0px;">
    Done. Have a Fun !
  </li>
</ol>
== Frequently Asked Questions ==

**Is your widget is responsive ?**

yes,this widgert is fully responsive and work in all browsers.

**Is there any option to hide empty categories?**

yes,you can set this on widget options.

**Do you support the installation ?**

yes, i can help to install if you need.Visit [Here][1]

**Is there any short code to embed the widget in  post or pages ?**

yes,You can use this short code <span style="color: #333333; font-family: Consolas, Monaco, monospace; font-size: 16px; line-height: 24px;"><strong>[wc-category-accordion]</strong></span>

For direct use in template file echo do_shortcode**('<span style="color: #333333; font-family: Consolas, Monaco, monospace; font-size: 16px; line-height: 24px;">[wc-category-accordion]'</span>)**
